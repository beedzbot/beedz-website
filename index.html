Commands

Slash commands use / — Prefix commands use . (customisable with /setprefix)

/helpGeneral

Shows all available commands with descriptions and usage examples.

Usage: /help [command?]
/pingGeneral

Checks the bot's current response latency and connection status.

Usage: /ping
/setprefixGeneral

Set a custom prefix for all prefix-based commands in your server. Default is .

Usage: /setprefix <prefix>
/serverinfoInfo

Displays detailed info about the server including member count, boosts, and creation date.

Usage: /serverinfo
/userinfoInfo

Shows information about a user — join date, roles, and account age.

Usage: /userinfo [@user?]
/avatarInfo

Returns a full-resolution version of a user's avatar.

Usage: /avatar [@user?]
/botinfoInfo

Shows information about Beedz — version, uptime, and server count.

Usage: /botinfo
/kickModeration

Kicks a member from the server with an optional reason that is logged.

Usage: /kick @user [reason?]
/banModeration

Permanently bans a user from the server. Supports duration-based bans.

Usage: /ban @user [reason?] [duration?]
/muteModeration

Temporarily mutes a user, preventing them from sending messages or speaking in voice.

Usage: /mute @user <duration> [reason?]
/purgeModeration

Bulk-deletes a number of messages from the channel. Optionally filter by a specific user.

Usage: /purge <amount> [@user?]
/warnModeration

Issues a warning to a user which is logged to the server moderation history.

Usage: /warn @user <reason>
/warningsModeration

View all warnings issued to a specific user in this server.

Usage: /warnings @user
/slowmodeModeration

Sets a slowmode delay on the current channel. Use 0 to disable.

Usage: /slowmode <seconds>
/lockModeration

Locks a channel so regular members cannot send messages.

Usage: /lock [channel?]
/unlockModeration

Unlocks a previously locked channel, restoring send permissions.

Usage: /unlock [channel?]
/invitesInvites

Check how many members a user has invited to the server — total, regular, bonus, and how many have left.

Usage: /invites [@user?]
/invites leaderboardInvites

Shows the top 10 inviters in the server ranked by total invites, with a medal for the top 3.

Usage: /invites leaderboard:True
/set-leaves-channelInvites

Set the channel where Beedz posts a message whenever a member leaves the server, with their name and the new member count.

Usage: /set-leaves-channel #channel
/8ballFun

Ask the magic 8-ball a question and receive a mystical response.

Usage: /8ball <question>
/rollFun

Rolls dice. Supports standard RPG notation like 2d6 or 1d20.

Usage: /roll <NdN>
/triviaFun

Starts a trivia round in the channel. First correct answer wins!

Usage: /trivia
/coinflipFun

Flips a coin — heads or tails.

Usage: /coinflip
.deletePrefix

Quickly purges a set number of messages from the channel. Shorthand for /purge.

Usage: .delete <amount>
.clearPrefix

Clears all messages from the last hour in the current channel.

Usage: .clear
.sayPrefix

Makes the bot send a message in the current channel as itself.

Usage: .say <message>
.announcePrefix

Sends a formatted announcement embed to the current or specified channel.

Usage: .announce <message>
.mutePrefix

Quick-mute shorthand for prefix users. Mutes a user for a given duration.

Usage: .mute @user <duration>
.unmutePrefix

Removes a timeout/mute from a user immediately.

Usage: .unmute @user
.ticketTickets

Creates a new support ticket channel for the user who ran the command.

Usage: .ticket [reason?]
.closeTickets

Closes the current ticket channel and saves a transcript.

Usage: .close [reason?]
/ticket-panelTickets

Creates a ticket panel embed with a button that users can click to open a ticket.

Usage: /ticket-panel <title> [description?]
/ticket-addTickets

Adds a user to the current ticket channel so they can view and respond.

Usage: /ticket-add @user
/ticket-removeTickets

Removes a user from the current ticket channel.

Usage: /ticket-remove @user
/set-welcome-channelWelcome

Set the channel where Beedz sends a welcome message every time a new member joins.

Usage: /set-welcome-channel #channel
/set-welcome-messageWelcome

Set a custom welcome message. Use {user} for mention, {server} for server name, {count} for member count.

Usage: /set-welcome-message <message>
/test-welcomeWelcome

Sends a test welcome message to the configured channel so you can preview how it looks.

Usage: /test-welcome
/set-booster-channelBoosts

Set the channel where Beedz sends a thank-you message and pings whoever boosted the server.

Usage: /set-booster-channel #channel
/test-boostBoosts

Sends a test boost message to the configured channel so you can preview how it looks.

Usage: /test-boost
/embedUtility

Opens a form to create a fully custom embed message — set the title, description, colour, footer and image.

Usage: /embed
/create-voiceVoice

Creates a "Join to Create" voice channel. When members join it they get their own private VC with a 4 person limit.

Usage: /create-voice [category?]
/vc-panelVoice

Opens the VC control panel with buttons to lock, unlock, ghost, reveal, claim, disconnect members, view info, and adjust the user limit.

Usage: /vc-panel
/playMusic

Play a song in your voice channel by name or YouTube URL. Automatically queues if something is already playing.

Usage: /play <song name or URL>
/skipMusic

Skips the currently playing song and moves to the next in the queue.

Usage: /skip
/stopMusic

Stops the music, clears the queue, and disconnects the bot from the voice channel.

Usage: /stop
/pauseMusic

Pauses the currently playing track.

Usage: /pause
/resumeMusic

Resumes a paused track from where it left off.

Usage: /resume
/queueMusic

Shows the current music queue with track names and positions.

Usage: /queue
/volumeMusic

Sets the playback volume from 1 to 100.

Usage: /volume <1-100>
/nowplayingMusic

Shows the currently playing track.

Usage: /nowplaying
/button-ticketUI Buttons

Posts a stylish embed with an interactive "Open Ticket" button. Members click it to instantly open a support ticket channel.

Usage: /button-ticket [title?] [description?]
/button-purchaseUI Buttons

Posts an embed with a "Purchase" button that DMs the server team. Great for product or service servers.

Usage: /button-purchase <item> [price?] [description?]
/button-directionsUI Buttons

Creates a navigation embed with buttons linking to channels, rules, roles or external URLs — like a server directory.

Usage: /button-directions [title?]
/button-rolesUI Buttons

Posts a self-role embed with clickable buttons that assign or remove roles when members click them.

Usage: /button-roles <role1> [role2?] [role3?] [title?]
/button-verifyUI Buttons

Posts a verification embed with a "Verify" button. Members click it to receive the verified role automatically.

Usage: /button-verify [role?] [channel?]
/button-announceUI Buttons

Send a rich embed announcement with optional action buttons such as "React", "Join Event", or "Learn More".

Usage: /button-announce <title> <message> [button-label?]
/button-giveawayUI Buttons

Starts a button-based giveaway embed. Members click "Enter" and the button shows a live counter of how many people have joined. The host draws a winner at any time.

Usage: /button-giveaway <prize> [description?]
/button-pollUI Buttons

Creates an interactive poll embed with up to 4 button choices. Results update live as members vote.

Usage: /button-poll <question> <opt1> <opt2> [opt3?] [opt4?]
/buttonUI Buttons

Attach a fully custom button to any existing embed or message. Choose the label, color (blue/green/red/grey), and an optional emoji. The button re-displays the embed to whoever clicks it.

Usage: /button <text> <message_id> [color?] [emoji?]
